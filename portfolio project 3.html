Project Title :- TheFruitwala.com

website link :- https://utkarshd2308.github.io/The-Fruit-Wala.com/
This is a modern, responsive web design created for thefruitwala.com—a demo online fruit retail platform aimed at customers, shops, and wholesale buyers. It features a clean homepage, interactive product catalog, shopping cart functionality, and a contact form, all with smooth animations for an engaging buying experience. Technologies used: HTML, CSS, and basic JavaScript. The main goal of this project was to demonstrate user-friendly UI design, responsive layouts for all devices, and simple backend handling for orders and contact requests—showcasing how fresh produce can be efficiently marketed and sold in a digital workflow.

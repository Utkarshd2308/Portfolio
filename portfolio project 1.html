Project Title :- Westyard Agro Website

website link :- https://utkarshd2308.github.io/westyardweb/
This is a modern, responsive website designed for Westyard Agro as a demo agricultural business platform. It features a clean homepage, interactive product gallery, and a dedicated contact section—showcasing quality agricultural products with user-friendly navigation. Technologies used include HTML, CSS, and basic JavaScript. The project’s goal was to demonstrate intuitive UI/UX, adaptive layout for all devices, and basic product/service presentation for agribusiness. Animation effects and component cards provide a smooth, professional experience, illustrating the capabilities of practical web design for agriculture.

Technologies used: HTML, CSS, Basic JavaScript.

The goal of this project was to demonstrate UI design, responsive layout, and basic backend form handling.
